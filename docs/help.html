üè≠ Machine Configuration

Machine Types:

  • PrintNC/Heavy Hobby: Sturdy DIY machines
  • Light Industrial: Commercial entry-level CNCs
  • Industrial: Heavy-duty production machines

üîß Tool Setup

Supported Tool Types:

  • Flat end mills (most common)
  • Ball end mills (3D contours)
  • Chamfer mills (edge finishing)
  • V-bits (engraving, chamfers)
  • Drill bits (hole making)
  • Face mills (large flat surfaces)

Key Parameters:

  • Diameter affects surface speed
  • Flute count impacts feed rate
  • Stickout affects deflection

üéØ DOC Override Feature

  • Toggle Switch: Enable/disable manual DOC control
  • Custom Input: Specify exact cutting depth in millimeters
  • Real-time Validation: Immediate warnings for unsafe values
  • Material Limits: Automatic checking against material maximums
‚ö†Ô∏è Important: When DOC override is enabled, monitor warnings carefully. Excessive depth can cause tool breakage, poor surface finish, or machine damage.
